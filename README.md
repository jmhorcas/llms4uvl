# llms4uvl
LLMs for UVL


# How to
1. Install ollama in Linux:
`curl -fsSL https://ollama.com/install.sh | sh`


2. Check/Activate/Deactivate ollama service in the system
`sudo systemctl status ollama`
`sudo systemctl start ollama`
`sudo systemctl stop ollama`

3. Run model:
`ollama run gemma3`

Choose model from [https://github.com/ollama/ollama](https://github.com/ollama/ollama)
